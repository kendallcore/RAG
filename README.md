# Cognitive Combat Engine

A high-performance React application built with Vite and TypeScript.

## 🚀 Features

- **Blazing Fast**: Powered by Vite for near-instant HMR.
- **Type Safe**: Full TypeScript integration for robust development.
- **Modern UI**: Built with React 18.

## 🛠️ Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd RAG
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Create a production-ready build:
```bash
npm run build
```
